🍓🐍 Snake Game
🕹️ 遊戲玩法
- 使用 **方向鍵 / WASD** 控制蛇的移動
- 吃到不同物件會產生不同效果：
  - 🍓 草莓
    - 分數 +10
    - 蛇身體變長
  -⭐ 星星
    - 進入 **3 秒無敵狀態**
    - 無敵期間：
      - 撞牆不會死亡（會從另一側出現）
      - 撞到自己也不會死亡
    - 畫面上方會出現 **無敵倒數條**
    - 無敵即將結束時，倒數條會 **閃爍提示**
- ✨ 主要特色
- 除了「存活 / 死亡」，加入「無敵狀態」
- **視覺回饋**：
  - 無敵倒數條
  - 倒數結束前閃爍提示
  - 無敵期間蛇的外觀變化
- **最高分紀錄**：
  - 最高分會被儲存在瀏覽器中
  - 重新整理或關閉頁面後仍可保留
